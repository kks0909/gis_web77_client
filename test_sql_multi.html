Проверка группы запросов

Файл:
Тип запроса:



Типы данных:




Произвольная <data>



Файл не загружен.


Руководство:
1) вводим название файла для проверки и нажимаем "Загрузить файл"
2 когда файл подгрузится, в желтом блоке под кнопкой "Выполнить" должен появиться текст "Файл загружен"
3) выбираем в блоке "Тип запроса" необходимые чекбоксы для проверки
4) розовый блок отвечает за значения передачу значений в переменные для каждого запроса:
- в поле "произвольная <data>" можно вписать свои параметры, переменные.
5) выставляем необходимые чекбоксы в зеленом блоке
- чекбокс "во все запросы вставлять WKT" - некоторые запросы неявно требуют передачу WKT геометрии(insert). Если чекбокс включен, то к запросу добавляются данные из розового блока поля WKT
- чекбокс "вставлять произвольную data" - вместо параметров, описанных в запросе, <data> будет заменяться на то, что указано в розовом блоке в поле "произвольная <data>"
- чекбокс "Использовать значения по умолчанию для параметров, если они есть" - указывает, нужно ли в параметры запроса передавать значения по умолчанию, или брать значения из розового блока
6) нажимаем кнопку "выполнить"
7) лог выполнения будет записываться в файл Log\WebSite\TESTSQL.txt
